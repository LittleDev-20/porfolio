# Portfolio (XAMPP-ready)

Open `index.html` in your browser, or run it through XAMPP by visiting:

- `http://localhost/porfolio/`

## Edit your info

Update your name, job history, skills, projects, and links in:

- `script.js`

## Add project images

- Put images in `assets/` and set `image: "assets/your-file.png"` on a project in `script.js`.
- For multiple screenshots, add `images: ["assets/x.png", "assets/y.png"]` (the page shows a **Screenshots** button).

## Add certificates

- Put certificate images in `assets/certificates/`
- Update `profile.certificates` in `script.js` to match your filenames
- If you see “Missing certificate image” in the Certificates section, it means the file is not in `assets/certificates/` yet.
